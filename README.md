# AIML Industry Radar

AI-powered job market intelligence system that analyzes real job postings and identifies the most in-demand skills for AI, ML, Data Science, Data Engineering, and Software Engineering roles.

## Features

- Extracts skills from real job descriptions using Gemini
- Normalizes skill names
- Aggregates industry demand trends
- Performs role-wise skill analysis
- Identifies skill gaps for users
- Generates personalized recommendations

## Tech Stack

- Python
- Gemini API
- JSON
- Git
- GitHub

## Project Pipeline

Raw Jobs
→ Skill Extraction
→ Skill Normalization
→ Aggregation
→ Role Analysis
→ Recommendation Engine
→ Dashboard Dataset
